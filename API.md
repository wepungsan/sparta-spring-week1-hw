# API 명세서
## 전체 게시글 조회
- METHOD : GET
- URL : /api/articles
## 특정 게시글 조회
- METHOD : GET
- URL : /api/article/{id}
## 게시글 작성
- METHOD : POST
- URL : /api/article
## 게시글 비밀번호 확인
- METHOD : POST
- URL : /api/article/{id}
## 게시글 수정
- METHOD : PUT
- URL : /api/article/{id}
## 게시글 삭제
- METHOD : DELETE
- URL : /api/article/{id}
